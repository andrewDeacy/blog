Multiple DevExpress Unbound Columns

December 15, 2014 by Andrew

DevExpress has two types of columns, bound and unbound. Bound columns are directly from the data source, while unbound columns can be calculated based off of other columns by handling the ColumnView.CustomUnboundColumnData event in which you can implement custom logic of any complexity.

This event iterates through each row of data for a data source bound to a given grid. Working with multiple unbound columns in each row of data can be difficult at first so I'll walk through it in this post.

After creating the event an .IsGetData check will determine if there is any data in the returned row.
